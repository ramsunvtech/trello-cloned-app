## API documentation

### Board Resources
  `GET` /boards , /boards/boardId  
  `POST` /boards  
  `PATCH` /boards/boardId  
  `DELETE` /boards/boardId  

### Column Resources
  `GET` /columns , /columns/columnId  
  `POST` /columns  
  `PATCH` /columns/columnId  
  `DELETE` /columns/columnId  

### Card Resources
  `GET` /cards , /cards/cardId  
  `POST` /cards  
  `PATCH` /cards/cardId  
  `DELETE` /cards/cardId
